# How to Generate an API Key in Race Results

Follow these step-by-step instructions to generate an API key in your RaceResult account. This API key is required to connect your RaceResult account with RaceLookup.

## Step 1: Log in to Your RaceResult Account

1. Go to [https://www.raceresult.com/](https://www.raceresult.com/)
2. Enter your Race Results credentials to log in

![RaceResult Login](/race-director/events/timing-companies/race-result-setup/RACE-RESULT-login.png)

## Step 2: Navigate to Account Access â†’ API Keys

1. In the left sidebar menu, locate and click on **"Account Access"**
2. From the Account Access submenu, click on **"API Keys"**
3. This will open the API Keys management page

![RaceResult Dashboard](/race-director/events/timing-companies/race-result-setup/RACE-RESULT-Dashboard.png)

## Step 3: View the API Keys Page and Create Form

1. You will see the API Keys page with the create form
2. The form is ready for you to fill in the required details

![API Keys Page and Create Form](/race-director/events/timing-companies/race-result-setup/RACE-RESULT-Create-API.png)

## Step 4: Enter API Key Details

1. **Enter a name for the API key**
   - In the name field, enter a descriptive name for your API key
   - Example: "RaceLookup Integration" or "My Race Event API Key"
   - This helps you identify the key later if you have multiple keys

2. **Select an expiration date for the key**
   - Choose an expiration date for your API key
   - This adds an extra layer of security by automatically expiring the key after a set period
   - You can select a date from the date picker or leave it blank for no expiration

![Enter API Key Details](/race-director/events/timing-companies/race-result-setup/RACE-RESULT-enter-details.png)

## Step 5: Copy Your API Key

1. After filling in all the required details, click the **"Create"** button to generate your new API key
2. Once the key is created, it will be displayed on the screen
3. **âš ï¸ IMPORTANT**: Copy the API key immediately and save it in a secure location
4. You will need to paste this key into the RaceLookup timing company configuration

![API Key Display](/race-director/events/timing-companies/race-result-setup/RACE-RESULT-api-key.png)

## Important Note

> **âš ï¸ The API key will only be displayed once.** If you refresh the page or close it, you will not be able to view the key again. Ensure you copy and store it safely at the time of creation. If you lose the key, you'll need to create a new one.

## Next Steps

After generating your API key:

1. Copy the API key to a secure location
2. Return to your RaceLookup account
3. Navigate to [Timing Companies configuration](./index)
4. Paste the API key into the API Key field
5. Save your configuration
